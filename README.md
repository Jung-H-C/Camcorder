메인화면
'M'키를 눌러 모드를 변경하거나 Space bar키를 눌러 녹화를 시작할 수 있습니다. ESC키를 누르면 프로그램이 종료됩니다.

반전모드
'M'키를 누르면 영상 명암이 반전이 됩니다. 



녹화화면
'Space bar'를 한번 더 누르면 녹화가 종료됩니다.
